1. Database & Core Infrastructure
Cloud Firestore (NoSQL Database)

How to use it: This is the core of your marketplace. You will use it to store and query your data structures.

Marketplace Examples: Storing product listings (titles, prices, descriptions, images), managing user profiles (buyers and sellers), and handling order histories.

Firebase Storage (File Storage)

How to use it: A secure bucket designed to hold massive files like images and videos.

Marketplace Examples: Allowing sellers to upload high-resolution product photos or digital downloadable assets (like PDFs, zip files, or software).

2. Communication & Real-Time Interaction
Firebase Cloud Messaging (FCM)

How to use it: Sends free, real-time push notifications across web browsers and mobile devices.

Marketplace Examples: Notifying a buyer when their order has shipped, or alerting a seller that someone bought their item or left a counter-offer.

Firestore Realtime Listeners

How to use it: A feature built into Firestore that pushes data updates to the client instantly without refreshing.

Marketplace Examples: Building a live buyer-to-seller chat system inside the marketplace, or updating stock counts instantly ("Only 1 item left!") while a user is looking at a listing.

3. Backend Logic & Integrations
Cloud Functions for Firebase (Serverless Backend)

How to use it: Lets you run secure, isolated backend Node.js code that automatically triggers based on events (like a user registering or a database document changing).

Marketplace Examples: * Integrating a payment gateway like Stripe to handle credit card processing securely.

Automatically calculating and adding sales tax.

Triggering an email receipt using a third-party service like SendGrid whenever a new order document is added to Firestore.

4. Search & Discovery (Extension)
Firebase Extensions (Algolia Search)

How to use it: Firestore doesn't natively support full-text search (like searching for misspelled words or partial terms). Firebase has a pre-built plugin that syncs your database with Algolia.

Marketplace Examples: Powering your marketplace search bar so users can filter items by keyword, category, price range, and location instantly.

5. Security & Maintenance
Firebase App Check

How to use it: Protects your backend resources from abuse, bots, and unauthorized API scrapping.

Marketplace Examples: Ensuring that malicious bots can't spam fake product listings, scrape your price data, or drain your database limits by executing infinite queries.

Firebase Crashlytics & Performance Monitoring

How to use it: Tracks bugs and loading speeds in real-time.

Marketplace Examples: Alerting you immediately if a checkout button stops working for users on a specific browser, or measuring if your product pages are loading too slowly.
